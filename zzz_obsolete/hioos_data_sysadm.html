Position Overview

The HiOOS Data System Administrator works within the Hawaiʻi Ocean Observing System (HiOOS) Data Management and Communications (DMAC) team to help design and develop a system to acquire and archive oceanographic data (remote sensing, models, observations from buoys, moorings, gliders, etc.) and to provide a system by which users can search and access these data. The data system will also integrate these data streams to facilitate the development of useful derived data products and delivery systems. We ensure that data formats are consistent with national standards; provide binary data transport via OPeNDAP, FTP/HTTP, Open Geosptial Consortium (OGC) services or other Web-based servers; develop Web-browsing capabilities through a Live Access Service (LAS), Web Map Server (WMS), Google Maps API, and other technologies; and work in coordination with Web developers and data assistants who are responsible for populating the HiOOS portal with products and data provided by HiOOS scientific investigators. HiOOS is part of the Pacific Islands Integrated Ocean Observing System (PacIOOS) Regional Association within the larger U.S. Integrated Ocean Observing System (IOOS), which contributes to the Global Ocean Observing System (GOOS), the oceanographic component of the Global Earth Observing System of Systems (GEOSS).

Duties

  1. Develop, implement and maintain a data system for the archive management of HiOOS data streams.
  2. Assist in the implementation of Web-based delivery mechanism(s) for HiOOS products including LAS and WMS, and specifically how these servers will interact with the data base.
  3. Provide a mechanism for integrating disparate data systems in compliance with the Integrated Ocean Observing System (IOOS) mandated data management procedures.
  4. Interact with HiOOS data management team to provide access to data and data products.
